# 2026 FIFA 世界杯 32强对阵预测

一个零依赖、纯前端的单页 Web 工具：填写 12 个小组的 1/2/3 名预测，自动推导 8 席最佳第 3 名，并实时生成 32强 → 16强 → 8强 → 半决赛 → 决赛的完整对阵图。

## 在线访问

GitHub Pages： https://Linjiababa001.github.io/worldcup2026-bracket/

## 功能

- **小组赛预测**（A–L 共 12 组）
  - 两步式交互：先点槽位（1/2/3/4）→ 再点球队 → 写入对应名次
  - 智能推荐按钮：按当前积分自动填入 1/2/3 名
  - 进度条：显示已填组数
- **8 席最佳第 3 名榜**：按 FIFA 排序规则（积分 → 净胜球 → 进球 → 胜场）实时排名，前 8 名晋级 R32
- **淘汰赛对阵图**：64 场完整结构（R32 → R16 → QF → SF → F），上下半区分屏
  - 缩放控件 + 缩略图导航
  - SVG L 型连接线
  - URL hash 锚点跳转
- **本地持久化**：所有预测自动保存到 localStorage（key: `worldcup2026_predictions_v1`）
- **响应式设计**：桌面 4 卡片并排，移动端单列

## 使用方式

1. 打开页面 → 默认进入 "小组赛预测" 标签
2. 12 个组按 A-D / E-H / I-L 三个子 tab 分屏
3. 点击右上角 1/2/3/4 按钮选中槽位（变蓝发光）
4. 点击球队行 → 该球队填入选中的名次
5. 填完 12 组后，切到 "淘汰赛对阵" 查看自动生成的对阵图
6. 随时刷新或关页面再开，预测数据不丢（localStorage）
